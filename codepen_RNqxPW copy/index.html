The majority of the internet is made up of HTML documents.


HTML - Hypertext Markup Language


TYPES OF FILES FOUND ON WEB:
PLAIN TEXT, HTML, IMAGES, VIDEO AND MUSIC

HTML is made up of:
text content, mark-up tags, references and links

Tags allow you to affect the output of the code that is written.
There are two types of tags inline and block.
Inline allows the code to run continuously, like we are doing now.

Whereas block contains the content within a virtual box with adjustable height and width.

HTML code also allows you to turn words into images.
